# Airtable Skills Test: HR Onboarding Dashboard

Welcome, and thank you for your interest in the Airtable Skills Test.

This repository contains the tasks for the Airtable HR Onboarding Dashboard assessment. The exercise is designed to evaluate your ability to use Airtable's features like formulas, automation, and interfaces to solve real-world HR challenges.

### Tasks
This test consists of two tasks:

- **Task 1: Intermediate Airtable Skills**: Work with formulas to clean, categorize, and track new hires.
- **Task 2: Advanced Airtable Automation**: Build an automation that triggers actions for "High Value" new hires and create a dashboard interface for HR visibility.

### Submission Process
1. **Fork this repository**.
2. Complete the tasks in your fork.
3. Add your files to the appropriate folders (`submissions/`, `assets/`).
4. Fill out the `SUBMISSION_TEMPLATE.md` file.
5. Open a Pull Request back to this repository.

Please refer to the **INSTRUCTIONS.md** and **TASK_1.md** / **TASK_2.md** for task details.

We look forward to reviewing your work!
